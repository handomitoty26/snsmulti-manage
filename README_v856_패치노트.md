# v8.5.6 패치노트 — 초고수 벤치마킹 분석센터

릴리스: 2026-07-12
기준 버전: v8.5.5 맛집·여행 후기 수익화 워크플로우판

## 새 메뉴

**콘텐츠 그룹 → 초고수 분석센터** (사이드바 ★ 아이콘)

기존 "벤치마킹" 탭은 그대로 유지됩니다. 초고수 분석센터는 그 위에 계정 단위 추적과 패턴 성과 검증을 얹은 상위 시스템입니다.

## 1차수 — 발굴 + 계정 후보 점수표

- **초고수 발굴 프롬프트 생성기** 3종
  - 초고수 검색 키워드 20개 생성 (카테고리·플랫폼 선택 반영)
  - 플랫폼별 발굴 루트 안내 (15분 내 탐색 동선 설계)
  - 30초 1차 필터 질문지
- **검색 바로가기** — 네이버 인플루언서, 네이버 통합검색, 인스타 탐색, Threads, YouTube
- **계정 후보 등록** — 계정명, 플랫폼, 링크, 카테고리, 추정 수익모델, 첫인상 메모, 상태
- **시그널 점수표** — 링크 구조 / 광고 표시 반복 / 저장 유도력 / 반복 공식 / 구매 의도 댓글 (각 0~20점)
- **자동 등급** — S(80+) / A(65+) / B(50+) / C(35+) / D
- **AI 점수화** — 관찰 자료 붙여넣기 → 프롬프트 복사 또는 ⚡ 원클릭 실행 → JSON 자동 저장
- **JSON 오류 수정 프롬프트** — 기존 트렌드 센터와 동일 방식

## 2차수 — 심층 해부 + 노하우 패턴 저장소

- **해부 프롬프트 생성기** — 계정·유형·제목·원문을 조합해 구조 분석 프롬프트 자동 조립
- **⚡ AI로 바로 해부** — API 키가 있으면 원클릭, 없으면 복사·붙여넣기 방식
- **추출 항목** — 제목공식, 후킹유형, 후킹공식, 본문골격, 수익동선, 신뢰장치, 광고표시방식, CTA공식, 반복가능성, 정책위험, **복제 금지 요소**
- **노하우 패턴 저장소** — 해부 결과가 패턴 단위로 분리 저장 (7개 유형: 제목공식/후킹공식/본문골격/수익동선/신뢰장치/CTA문구/기타)
- **원문 복제 자동 차단** — 패턴 공식이 원문 문장과 일치하면 저장 시 자동으로 걸러지고 차단 건수가 표시됩니다
- 패턴 직접 추가 / 유형·카테고리 필터 / 삭제 지원
- 원문은 200자 발췌만 보관 (전문 저장 안 함)

## 3차수 — 우리식 변형 생성기

- 패턴 + 내 상품DB 소재 + 연결 캠페인 + 광고/제휴 상태 선택
- **무단 복제 방지 체크리스트 6개** — 모두 통과해야 저장 가능 (미통과 시 저장 차단)
- **변형 프롬프트 자동 조립** — 내 말투 프로필(AI 연동 설정), 경험 상태, 표시문구, 복제 금지 요소가 프롬프트에 자동 반영
- **⚡ AI로 바로 변형** → Threads / 카드뉴스 / Facebook / Blog 4채널 초안 생성
- **게시계획 + 카드뉴스 + 링크허브 일괄 저장**
  - 표시문구가 글 첫 부분에 자동 삽입
  - 정책검수 안전 점수 자동 계산
  - 캠페인 미선택 시 `[패턴실험]` 캠페인 자동 생성
  - **모든 저장 콘텐츠에 patternId 태깅** (5차수 집계의 열쇠)
- 미사용·미방문 소재는 경험 상태가 "자료 조사/후기 정리"로 자동 고정

## 4차수 — 패턴별 성과 리포트 + 승리 공식 저장소

- **패턴별 성과 집계** — patternId로 묶어 글 수, 성과기록 수, 조회, 클릭, 저장, 공유, 구매/문의, 순수익, 수익 점수를 자동 계산
- 정렬 기준: 순수익 → 클릭 (조회수 아님)
- **패턴 리포트 CSV 내보내기**
- **승리 공식 승격** — 성과 검증된 패턴을 승리 공식 저장소로 승격 (성과 스냅샷과 승격 근거 자동 기록)
- **승리 공식 재투입** — [변형에 사용] 버튼으로 변형 생성기에 바로 연결
- 승격 시 성과 기록이 없으면 확인 후 수동 승격 가능 (근거에 "성과 검증 전"으로 표시)

## 데이터 구조 변경

`dataVersion: 8.5.5 → 8.5.6`

추가된 localStorage 상태:

```
expertCandidates  계정 후보 + signals{linkStructure, disclosure, saveHook, repeatFormula, purchaseComments}
expertContents    해부 콘텐츠 + analysis{titleFormula, hookType, bodyFlow, revenuePath, doNotCopy, ...}
knowhowPatterns   패턴 저장소 (type, formula, category, sourceContentId, useCount)
winningFormulas   승리 공식 (patternId, statsSnapshot, reason, promotedAt)
```

기존 배열 확장:

- `posts[].patternId` (빈 문자열로 추가)
- `cardNews[].patternId` (빈 문자열로 추가)

- 기존 캠페인·소재·글·트렌드·벤치마킹 사례·설정·AI 키 설정 모두 유지됩니다.
- **업데이트 전 반드시 JSON 백업을 먼저 저장하세요.**

## 유지된 안전 원칙

- 자동 게시, 자동 로그인, 무단 크롤링, 플랫폼 API 자동 연동 없음
- 사용자가 직접 본 공개 콘텐츠만 입력
- AI가 계정 이름·팔로워 수·수익 금액을 지어내지 않도록 모든 프롬프트에 안전 원칙 명시
- 수익 규모는 "추정"으로만 표기
- 원문 복제 차단 + 체크리스트 게이트 + 표시문구 자동 삽입 + 정책검수 연동
