# v8.5.5 패치 노트

## 맛집·여행 후기 수익화 워크플로우판

### 추가 기능
1. 여행 카테고리 추가
2. 상품DB 세부 유형 필드 추가
3. 지역·역명·여행지 필드 추가
4. 방문일·여행일 필드 추가
5. 사진 보유 상태 필드 추가
6. 맛집 후기 자동 구성 버튼 추가
7. 맛집 후보 정리 버튼 추가
8. 여행 코스 글쓰기 버튼 추가
9. 여행 준비물 글쓰기 버튼 추가
10. 여행 준비물 링크허브 자동 준비
11. 맛집·여행 이미지 플랜 자동 연결
12. CSV 양식 확장

### 정책 안전 강화
- 직접 방문하지 않은 맛집은 후보 정리형으로 작성하도록 안내
- 여행 준비물 제휴형 글은 표시문구와 링크허브 중심 연결을 권장
- 미방문/미사용 콘텐츠는 정보형 이미지와 안전 문구를 우선 사용
