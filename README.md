# 멀티 SNS 수익화 관리자 Pro v8.5.3

## 터보 워크플로우 & 네이버 블로그 품질 강화판

이번 버전은 실전 사용 흐름을 줄이고, 네이버 블로그 애드포스트 승인 준비에 맞는 정보성 글 작성을 강화한 패치입니다.

## 주요 변경 사항

### 1. 상품DB → 캠페인 자동 생성
상품·맛집DB 목록에 다음 버튼이 추가되었습니다.

- **캠페인 만들기**: 상품DB 정보로 캠페인 초안을 자동 생성하고 수정 화면으로 이동
- **캠페인+글쓰기**: 캠페인 자동 생성 후 멀티SNS 글쓰기 화면으로 이동
- **블로그 품질 글쓰기**: 네이버 블로그 정보성 글 화면으로 이동

자동 생성 캠페인에는 상품명, 카테고리, 수익 방식, 경험 상태, 장점, 단점, 추천 대상, 주의 표현, 운영 방향이 메모로 자동 정리됩니다.

### 2. 캠페인 수정 기능
캠페인 목록에 **수정** 버튼이 추가되었습니다. 자동 생성된 캠페인도 언제든 캠페인명, 목표, 기간, 메모를 수정할 수 있습니다.

### 3. 중복 캠페인 방지
같은 상품DB 항목으로 만든 캠페인이 이미 있으면 기존 캠페인을 사용할지, 새 캠페인을 만들지 선택할 수 있습니다.

### 4. 네이버 블로그 품질 모드
블로그·협찬 탭에 다음 기능이 추가되었습니다.

- **네이버 품질 목차**
- **AI 블로그 고품질 초안**
- **품질 점수 확인**

블로그 품질 점수는 정보성, 구체성, 경험/자료 기준 표현, 과장 표현 안전성, 문단 구성, 광고성 완화 기준으로 계산됩니다.

### 5. ChatGPT API 400 오류 개선
OpenAI 호출 방식을 모델별로 분리했습니다.

- `gpt-4o-mini`, `gpt-4.1-mini`: 기존 Chat Completions 방식
- `gpt-5.4-mini`, `chat-latest`: Responses API 우선 방식

400 오류가 발생하면 단순 “서버 오류”만 보이지 않고 실제 오류 상세 일부를 함께 보여줍니다.

### 6. 블로그 전용 AI 모델 설정
설정 → AI 연동에 **블로그 고품질 모델** 선택 항목이 추가되었습니다.

권장 조합:

- 트렌드 수집: Gemini
- SNS 짧은 글 / 정책검수: `gpt-4o-mini`
- 네이버 블로그 품질 글: `gpt-5.4-mini`

## 업데이트 후 확인 사항

1. GitHub Pages에 파일 업로드
2. 앱을 새로 열기
3. 하단 또는 화면에 업데이트 안내가 뜨면 새로고침
4. 안 되면 브라우저 새로고침 또는 앱 재설치
5. 기존 데이터는 localStorage에 남아 있지만, 업데이트 전후 JSON 백업을 권장합니다.

## 사용 흐름

### 일반 SNS 수익형

AI 트렌드 자동 수집 → 상품DB 등록 → 상품DB에서 캠페인+글쓰기 → 4개 채널 글 만들기 → AI 다듬기 + 정책검수 → 게시계획 저장 → 직접 게시 → 성과 기록

### 네이버 블로그 품질형

AI 트렌드 자동 수집 → 상품DB 등록 → 상품DB에서 블로그 품질 글쓰기 → 네이버 품질 목차 → AI 블로그 고품질 초안 → 품질 점수 확인 → 직접 검수 후 블로그 게시
