# 멀티 SNS 수익화 관리자 Pro v8.5.5

## 맛집·여행 후기 수익화 워크플로우판

이번 버전은 맛집 글과 여행 후기를 수익화 흐름에 맞게 작성할 수 있도록 프로그램 워크플로우를 강화한 버전입니다.

## 핵심 추가 기능

### 1. 여행 카테고리 추가
상품·맛집DB, 캠페인, 글쓰기, 카드뉴스, 트렌드, 리포트 카테고리에 `여행`을 추가했습니다.

### 2. 상품·맛집DB 입력 필드 확장
다음 필드가 추가되었습니다.

- 세부 유형: 맛집 방문 후기, 맛집 후보 정리, 여행 코스, 여행 준비물, 숙소, 체험/관광지 등
- 지역·역명·여행지
- 방문일·여행일
- 사진 보유 상태: 직접 사진 충분, 직접 사진 일부, 사진 없음, 정보형 이미지만 사용 등

### 3. 맛집·여행 자동 글쓰기 버튼
상품DB 목록에 다음 버튼이 추가되었습니다.

- 맛집 후기
- 맛집 후보
- 여행 코스
- 여행 준비물

버튼을 누르면 블로그 탭에 글 구조가 자동 생성되고 AI 이미지 플래너까지 함께 연결됩니다.

### 4. 여행 준비물 링크허브 자동 준비
`여행 준비물` 버튼 사용 시 여행 준비물 체크리스트 링크허브 항목을 자동으로 준비합니다.

### 5. 이미지 플래너와 자동 연결
맛집/여행 경험 상태, 사진 보유 상태, 카테고리에 따라 직접 촬영 중심 또는 정보형 이미지 중심으로 이미지 패키지를 자동 제안합니다.

### 6. CSV 양식 확장
CSV 대량 입력 양식에 세부 유형, 지역, 방문일, 사진 상태 필드가 추가되었습니다.

## 추천 사용 흐름

### 맛집 직접 방문 후기
상품DB 등록 → 맛집 후기 → 블로그 초안 확인 → 이미지 플랜 → 카드뉴스 제작소 → 협찬 제안서 → 성과 기록

### 미방문 맛집 후보 정리
상품DB 등록 → 맛집 후보 → 직접 방문 표현 제거 확인 → 정보형 이미지 카드 → 블로그 게시 → 방문 후 업데이트

### 여행 코스 후기
상품DB 등록 → 여행 코스 → 동선/비용/준비물 구조 확인 → 이미지 플랜 → 카드뉴스 → 링크허브 연결

### 여행 준비물 제휴형
상품DB 등록 → 여행 준비물 → 링크허브 항목 확인 → 제휴 표시문구 확인 → 블로그/카드뉴스 게시

## 포함 문서
전체 패키지 ZIP에는 맛집·여행 후기 수익화 작성 가이드 DOCX/PDF, 사용자 매뉴얼, 운영계획서, 프로젝트 지침이 함께 포함됩니다.
