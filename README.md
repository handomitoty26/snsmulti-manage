# 멀티 SNS 수익화 관리자 Pro v8.5.6

## 초고수 벤치마킹 분석센터판

이번 버전은 상위 수익자(초고수)의 SNS 수익화 노하우를 **찾고 → 점수 매기고 → 구조를 해부하고 → 우리식으로 변형하고 → 성과로 검증해 승리 공식으로 저장**하는 5단계 파이프라인을 프로그램에 내장한 버전입니다.

## 왜 이 구조인가

AI에게 "고수익 SNS 계정을 찾아줘"라고 하면 **존재하지 않는 계정을 지어낼 수 있습니다.** 또한 이 프로그램은 무단 크롤링과 자동 수집을 하지 않습니다.

그래서 v8.5.6은 역할을 이렇게 나눕니다.

- **AI가 하는 일**: 검색 전략 설계, 검색 키워드 생성, 후보 점수화, 콘텐츠 구조 해부, 우리식 변형, 성과 집계
- **사람이 하는 일**: 실제 검색으로 계정 후보 찾기(10~15분), 공개 콘텐츠 붙여넣기, 최종 검수와 직접 게시

## 5단계 파이프라인

### 1단계 · 초고수 발굴
AI가 검색 키워드 20개, 플랫폼별 발굴 루트, 30초 판별 질문지를 만들어 줍니다. 검색 바로가기 버튼(네이버 인플루언서, 인스타 탐색, Threads 등)으로 바로 이동합니다.

**진짜 초고수 시그널 5가지 (팔로워 수가 아닙니다)**

1. 링크 구조 — 프로필 링크허브에 제휴/스토어/협찬 문의가 정리되어 있는가
2. 광고 표시 — [협찬] [제휴] 표시 게시물이 반복되는가 (실제로 돈을 받는 증거)
3. 저장 유도력 — "저장해두세요" 류 콘텐츠 비율
4. 반복 공식 — 제목·구조가 템플릿처럼 반복되는가
5. 구매 의도 댓글 — "어디서 사요?", "링크 주세요" 비율

### 2단계 · 계정 후보 점수표
시그널 5개를 각 20점으로 채점해 S/A/B/C/D 등급을 자동 산출합니다. 직접 입력하거나, 관찰 자료(프로필 소개 + 대표 캡션 + 메모)를 붙여넣고 AI에게 점수화를 맡길 수 있습니다. **S·A 등급만 3단계로 넘기고 B 이하는 버립니다.**

### 3단계 · 콘텐츠 심층 해부
S급 계정의 대표 콘텐츠를 1건씩 해부해 노하우 DNA를 추출합니다.

- 제목 공식 (변수형: `{대상} 사기 전에 꼭 확인할 {숫자}가지`)
- 첫 3초 후킹 유형 (공포형/궁금형/공감형/숫자형/비교형)
- 본문 골격 (문제제기 → 실수 → 체크포인트 → 추천 대상 → 링크)
- 수익 동선, 신뢰 장치, 광고 표시 방식
- **복제 금지 요소** — 그대로 쓰면 표절이 되는 고유 문구를 AI가 명시

추출된 패턴은 **노하우 패턴 저장소**에 유형별로 쌓입니다. 맛집에서 발견한 제목 공식을 전자기기 캠페인에 재사용할 수 있습니다.

### 4단계 · 우리식 변형 생성기
패턴의 **구조만 빌리고 문구는 100% 새로** 씁니다. 내 상품DB 소재, 내 말투 프로필, 내 경험 상태(직접 사용/자료 조사)가 자동 반영되고 표시문구가 첫 부분에 삽입됩니다. 결과는 Threads / 카드뉴스 / Facebook / Blog 4채널로 나와 게시계획·카드뉴스·링크허브에 저장됩니다.

### 5단계 · 패턴별 성과 리포트 + 승리 공식
변형 콘텐츠에는 **패턴 ID가 태그**됩니다. 게시 후 성과를 입력하면 패턴 단위로 클릭·저장·구매·순수익이 자동 집계됩니다.

성과가 검증된 패턴을 **승리 공식**으로 승격하면, 이 순간부터 남의 노하우가 아니라 **내 데이터로 검증된 내 공식**이 됩니다. 승리 공식은 변형 생성기에 바로 다시 투입할 수 있습니다.

## 안전 원칙 (v8.5.6에서도 유지)

- 자동 크롤링·자동 수집·자동 게시 없음. 직접 본 공개 콘텐츠만 붙여넣습니다.
- **원문 복제 자동 차단**: 해부 결과의 패턴 공식이 원문 문장과 같으면 저장이 자동으로 걸러집니다.
- 변형 저장 전 무단 복제 방지 체크리스트 6개를 모두 통과해야 합니다.
- 미사용·미방문 소재는 경험 상태가 "자료 조사/후기 정리"로 자동 고정되어 직접 경험처럼 쓰이지 않습니다.
- 수익 규모는 "추정"으로만 표기하며 사실처럼 저장하지 않습니다.
- 제휴/협찬 표시문구가 글 첫 부분에 자동 삽입되고, 정책검수 점수가 함께 계산됩니다.

## 데이터 구조

localStorage에 4개 상태가 추가되고 `dataVersion`이 **8.5.6**으로 올라갑니다.

- `expertCandidates` — 계정 후보 + 시그널 점수 + 등급
- `expertContents` — 해부한 대표 콘텐츠 (원문은 200자 발췌만 보관)
- `knowhowPatterns` — 노하우 패턴 저장소
- `winningFormulas` — 성과 검증된 승리 공식

기존 v8.5.5 데이터(캠페인, 소재, 글, 트렌드, 벤치마킹 사례, 설정)는 **손실 없이 자동 마이그레이션**됩니다. 기존 글에는 `patternId` 필드가 빈 값으로 추가됩니다.

## 추천 사용 흐름

### 주말 1회 · 초고수 발굴 루틴 (약 30분)
초고수 분석센터 → 1단계 [검색 키워드 20개 생성] → 직접 검색 10분 → 후보 5~10개 등록 → 2단계 AI 점수화 → S·A 등급 2~3명 선별

### 평일 · 노하우 흡수 루틴 (약 20분)
S급 계정 → [해부하기] → 대표 콘텐츠 캡션 붙여넣기 → [⚡ AI로 바로 해부] → 패턴 저장소에 공식 축적

### 콘텐츠 제작 루틴
패턴 선택 → 내 소재 선택 → 체크리스트 6개 확인 → [⚡ AI로 바로 변형] → 검수 → 게시계획 저장 → 직접 게시

### 검증 루틴 (2주 후)
성과분석에 클릭·구매·수익 입력 → 5단계 리포트에서 패턴별 비교 → 상위 패턴 [승리 공식 승격]

## 기존 기능

트렌드 수집, 상품·맛집DB, 멀티SNS 글쓰기, 카드뉴스 제작소, AI 이미지 플래너, 맛집·여행 워크플로우, 링크허브, 성과분석, 리포트, 정책검수, 미디어킷, 이중 AI 연동(Gemini + ChatGPT)은 모두 그대로 유지됩니다.
