트렌드 운영 원칙

조회수보다 클릭, 저장, 구매/문의, 순수익, 협찬 가능성을 우선합니다.
직접 써보지 않은 상품은 “후기와 상품 정보를 기준으로 정리했습니다”처럼 표현합니다.
제휴/협찬/제품 제공 가능성이 있으면 첫 줄 표시문구를 사용합니다.
자동 크롤링, 자동 게시, 자동 로그인, ChatGPT/Gemini API 연동은 사용하지 않습니다.
