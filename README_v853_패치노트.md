# v8.5.3 패치노트 — 터보 워크플로우 & 네이버 블로그 품질 강화판

## 핵심 요약

이번 패치는 사용자가 실전 테스트 중 발견한 반복 작업을 줄이고, ChatGPT API 모델별 호출 오류를 개선하며, 네이버 블로그 애드포스트 승인 준비에 맞는 정보성 글 작성 흐름을 강화한 버전입니다.

## 1. 상품DB → 캠페인 자동화

상품·맛집DB 목록에 다음 버튼을 추가했습니다.

- 캠페인 만들기
- 캠페인+글쓰기
- 블로그 품질 글쓰기

상품DB 정보로 캠페인명, 카테고리, 목표, 목표 클릭, 목표 수익, 기간, 메모가 자동 생성됩니다.

## 2. 캠페인 수정 기능

캠페인 목록에 수정 버튼을 추가했습니다. 자동 생성된 캠페인도 나중에 얼마든지 캠페인명, 목표, 메모, 기간을 수정할 수 있습니다.

## 3. 중복 캠페인 방지

동일 상품으로 기존 캠페인이 있으면 기존 캠페인을 사용할지, 새 캠페인을 만들지 선택할 수 있습니다.

## 4. 네이버 블로그 품질 모드

블로그·협찬 탭에 다음 기능을 추가했습니다.

- 네이버 품질 목차
- AI 블로그 고품질 초안
- 품질 점수 확인

이 모드는 제휴 링크 직접 삽입과 강한 구매 유도보다 정보성, 구체성, 장점/단점 균형, 경험/자료 기준 표현을 우선합니다.

## 5. 블로그 품질 점수

100점 기준으로 다음 항목을 점검합니다.

- 본문 길이
- 구매 전 기준과 체크포인트
- 장점 설명
- 단점/주의점
- 경험/자료 기준 표현
- 과장 표현 안전성
- 구매 유도 완화

## 6. ChatGPT API 400 오류 개선

OpenAI 모델별 호출 방식을 분리했습니다.

- gpt-4o-mini / gpt-4.1-mini: 기존 Chat Completions 방식
- gpt-5.4-mini / chat-latest: Responses API 우선 방식

오류 발생 시 실제 오류 상세 일부를 표시하도록 개선했습니다.

## 7. 블로그 전용 AI 모델 설정

설정 → AI 연동에 블로그 고품질 모델 선택 항목을 추가했습니다.

권장값:

- 기본 ChatGPT 모델: gpt-4o-mini
- 블로그 고품질 모델: gpt-5.4-mini
